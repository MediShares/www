在区块链上建立互信世界

什么是 Mutual DAO System
  • •
    基于区块链的、去中心化的、开源的相互保障合约市场
  • •
    任何人可以通过锁定 Mutual DAO System 的 MDS Token 获取平台上的相互保障合约的赔付资格，从而获得合约所指定的保障
  • •
    任何人可以通过 Mutual DAO System 平台和 Mutual DAO System 提供的智能合约模板发布相互保障合约， 并从中获得 MDS奖励
Mutual DAO System 的优势
  • 无资金池： 用智能合约解决资金安全问题
  • 高效运营： 用智能合约和 DAO 提升相互保障合约运营效率
  • 激励机制： 任何人都可以基于智能合约模板发起一个共享保障合约， 并因此获得 MDS Token
  • 价值共享： 让保障参与者能够分享社区所产生的收益和价值
  • 全球市场： 基于 MDS Token 的无国界流通特性， Mutual DAO System 市场对全球用户开放， 并用区块链完成赔付和清算
应用场景
  • 大病保障合约
    大病保障合约主要涵盖癌症等重大疾病的保障，通过理赔调查公示和社区投票完成赔付，Mutual DAO System 能够很好的解决大病保障的资金池透明度问题
  • 风险运动爱好者保障合约
    全球有大量潜水爱好者，但潜水作为死亡率最高的风险运动却没有相应的意外伤害保险产品，Mutual DAO System 能够为社群相互保障提供一站式平台
  • 豪车车主保障合约
    豪车因为维修价格极高，大多数保险公司拒保，或者保险费用极高， 通过 Mutual DAO System 能够有效降低豪车保障的成本
  • 匿名疾病保障合约
    大量慢性疾病患者（如乙肝等）不愿意购买保险，因为担心可能会因此泄露个人隐私，丢掉工作。Mutual DAO System 能够通过区块链解决隐私安全的问题。
  • 天气灾害保障合约
    天气灾害在特定国家和地区内是非常小概率事件，但在全球范围则是时常发生的事件，通过读取全球天气和地震相关数据接口，Mutual DAO System 将实现真正意义上的全自动保障合约。
  • 虚拟资产保障合约
    随着虚拟数字资产越来越多的进入普通人的生活，对虚拟资产的安全保障需求也越来越强，Mutual DAO System 为虚拟资产类保障提供可能
保障合约市场
  • 只有符合 Mutual DAO System 标准的保障合约才能在市场上架，这些标准包括且不限：
    • •
      遵守 Mutual DAO System 的社区治理条例
    • •
      遵守 Mutual DAO System 统一约定的手续费标准
    • •
      能通过 Oracle 或者 DAO 判定赔付结果
  • 合约发起方将获得一部分 MDS 手续费作为奖励
  • 创建保障合约除了需要支付以太坊 Gas 外，还需要抵押一部分 MDS
